# JSON Formatter

A clean, high-tech JSON formatter built with Next.js, TypeScript, Tailwind CSS, and Lucide icons.

## Features

- Dark glass-style interface with responsive desktop and mobile layouts
- Local JSON parsing in the browser
- Syntax-highlighted JSON tree output
- Collapsible objects and arrays
- Real-time character count and validation status
- Accessible focus states and reduced-motion support

## Tech Stack

- Next.js 16.2 App Router
- React 19
- TypeScript
- Tailwind CSS 4
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```
